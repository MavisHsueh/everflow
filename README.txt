# Everriver GitHub Pages Starter

把這個資料夾整包上傳到 GitHub Pages（或任何靜態主機）即可看到「水紋過場」背景。

## 檔案結構
- `index.html`：首頁（含 Three.js Shader 水紋過場）
- `assets/water-displace.png`：擬真水紋位移貼圖（請自行放入）
- `videos/everflow-background1~6.mp4`：你的 6 支背景影片

## 使用步驟
1. 把你的 `water-displace.png` 放到 `assets/`。
2. 把 6 支影片放到 `videos/`，確保檔名為 `everflow-background1.mp4`~`everflow-background6.mp4`。
3. 直接用 `index.html` 在 GitHub Pages 發佈即可。
